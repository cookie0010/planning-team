소프트앤(Softn)의 제품 기획/전략 담당자 (sg.been2017@softn.kr).

- 앵커 서비스 퀴즈앤(QuizN, 교육·퀴즈 도메인)을 보유, 12개월 내 5개+ 신규 서비스 출시 계획.
- distribution(배포)·플랫폼 레버리지 관점에서 사고함. 개별 제품 매력도보다 "어떻게 기존 사용자 기반을 신규 서비스로 흘려보낼까"를 우선 프레임으로 삼는다.

- 설명이 추상적이면 "쉽게 풀어 달라"고 요청 — 구체적이고 평이한 비유 설명을 선호.
