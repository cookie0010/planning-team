quizn-board-v2 = AIDT 붕괴로 유휴된 1:1 교실 기기 위에 canvas형 실시간 공동참여 보드를 퀴즈앤 채널로 진입시키는 PoC 기획. 산출물 dir = docs/quizn-board-v2/.

버전 상태: writer N=4 (one-pager-v4.md, 2026-06-10). 리뷰어 최신 n=1 → one-pager-v4.1.md 저장 완료(2026-06-10). 다음 리뷰는 v4.2. (이전: writer N=2 → 리뷰 v2.1.) v4는 9섹션 — "5. 차별점"이 3.제안에서 분리됨(merge 금지).

**Why:** quizn-board v1 라인과 별개로 v2(canvas형)로 분기됨. v1 보드는 wall형, v2는 canvas형으로 제품 라인 분리가 핵심 논점.

**How to apply:** 이 프로젝트 리뷰 시 절대 손대면 안 되는 verbatim 보호 요소 —
- 1절 "현상 ≠ 문제 → 가설" 정직성 프레이밍, 3절 "검증된 해결책 아니라 가설적 제안 + 논리 도약" 불릿, 5절 "(가설)" 표기: 확신형으로 매끄럽게 만들지 말 것(honesty marker 풀강도 유지).
- 숫자: 기기 중 0.9/고 1.0, AIDT 8.1%/60%/32%→19%, 경기 120만, 첫 세션 10%, 2회+ 반복.
- (잠정) 태그(1차 퍼널 10%)는 그대로.
- 4절 벤치마킹 라이브 링크 전부 생존(Miro·FigJam·알로 + 출처: 서울경제·교육을비추다·THE AI·focusnjn·시사저널·알로블로그), 의도적 제외 note 유지.
- 8절 검증 질문 1차/2차 + 기각 기준 풀강도.

v4 리뷰 관찰: writer가 이미 어투 정리 완료, 손댈 것 거의 없음. 유일한 패턴 = 정직성 불릿의 서술형 종결("~아니다", "~제안이다")을 honesty 약화 없이 명사형(현상/제안)으로 도치.
